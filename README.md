# tinify-helper

一款使用tinify api压缩图片的插件

本插件使用[tinypng](https://tinypng.com/)API压缩图片,使用之前请先注册申请key值,并填入配置

## 配置

**tinify-helper.key**
  - 类型: string
  - 默认: ""

```
  "tinify-helper.key": "xxx",
```

## 例子

选中图片文件夹或者图片(类型`jpg`,`jpeg`,`png`),右键选择压缩选项(`tinifyImage`/`tinifyImages`)
